
n3zqx2l is a minimalist procedural and imperative systems programming language.


features of n3zqx2l:
========================

	- has no syntax.

	- has powerful dependent and computational type system.

	- has compile time function evaluation

	- has almost complete control over the LLVM IR language.

	- has typed hygenic macros.

	- has utf8 unicode character support.

	- 











things this language does NOT have:
====================================


	- no namespaces.

	- no comments.

	- no strings.

	- no character literals.

	- no builtin numeric constants.

	- no mathematical operators.
